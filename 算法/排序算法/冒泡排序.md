## 冒泡排序（Bubble Sort）

**思路：**

- 重复遍历数组
- 每次比较相邻两个数
- 大的往后挪动，像气泡一样往上冒
- 每轮确定一个最大数的位置

```c++
#include<iostream>

using namespace std;

void bubbleSort(int arr[], int len){
    // 外层循环：每一轮比较
    for(int i = 0;i < len - 1; i++){
        for(int j = ;j < len - i - 1;j++){
            // 比较大小，进行交换/冒泡
            if(arr[j] > arr[j + 1]){
                int temp = arr[j + 1];
                arr[j + 1] = arr[i];
                arr[i] = temp;
            }
        }
    }
}
```
