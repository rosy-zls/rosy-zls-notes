## 插入排序

### 思路

- 把数组分成已排序的部分 + 未排序的部分
- 每次拿未排序的第一个数
- 往已排序不分离找到合适位置插入
- 像整理扑克牌一样

```c++
#include <iostream>

using namespace std;

        // 待插入的数
        int key = arr[i];
        // 已排序部分的最后一个下标
        int j = i - 1;
        while( j >= 0 && arr[j] > key){
            arr[j + 1] = arr[j];
            j--;
        }
        // 插入到正确位置
    }
}
```
