## 防抖和节流

### 一、防抖

#### 1.1 什么是防抖

防抖就是在短时间多次触发同一事件时，只执行最后一次，从而避免频繁请求，降低服务器端的压力🍐。

#### 1.2 防抖的实现

```javascript
let debounce = (fn, delay) => {
    let timer = null;
    return function(...args){
        context = this;
        clearTimeout(timer);
        timer = setTimeout(()=>{
            fn.apply(context, args);
        },delay)
    }
}
```

#### 1.3 防抖的使用场景

- **实时搜索框：** 当用户在输入框输入内容的时候，等用户“停止”输入之后才会触发搜索。



### 二、节流

#### 2.1 什么是节流

节流就是在一段时间间隔内只触发一次，从而避免短时间内频繁触发，给服务器端降低压力🍐。
